### 自由の助手

1. 创建小程序组件、小程序页面，自动生成wxml、js、wxss、json文件、自动追加页面路径到app.json的pages中
2. 非微信原生标签高亮、wxml格式化可配置
3. 支持微信API代码块、es6语法代码块，并添加了简写对照
   
- 后续有望新增jq、layui的代码块
